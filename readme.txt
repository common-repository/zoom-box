== Description ==

This plug-in will place a small stylish [Zoom-Box](http://thesky.asia/zoom-box/ "Zoom-Box") widget at the top of your blog. 
The widget can used to zoom your blogs font size.

== Installation ==

1. Download and extract the plugin.
2. Upload `zoom-box` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. No more step. It just works.
